# Security Policy

## Supported Versions

Maclipp is currently pre-1.0. Security fixes are applied to the latest code on the default branch.

## Reporting A Vulnerability

Do not open a public issue for a vulnerability involving clipboard disclosure, local file access, shortcut injection, or unintended data retention.

Use GitHub's private vulnerability reporting feature for this repository. Include reproduction steps, affected macOS version, expected behavior, and the potential impact. Please allow reasonable time for investigation before public disclosure.

## Sensitive Data

Maclipp is a clipboard manager and may retain secrets copied by the user. Reports involving sensitive clipboard contents should use synthetic data and should not attach real credentials, tokens, or personal information.

Maclipp skips pasteboards marked concealed, transient, or autogenerated, enforces payload-size limits, and stores history with owner-only filesystem permissions. These controls reduce exposure but cannot identify every secret; source applications must provide sensitive markers, or users must pause recording manually.
